[ImageChoiceAndResize]

-------------------------------------------
このアプリケーションは、以下の処理を行います。

1.学習モデル画像素材の不良素材の選択排除。

2.学習モデル画像素材の一括リサイズ。

3.学習モデル画像素材のパッケージ出力。

-------------------------------------------
This application performs the following processes:
1. Selects and eliminates defective image materials for the training model.
2. Bulk resizes training model image materials.
3. Packages and outputs training model image materials.
